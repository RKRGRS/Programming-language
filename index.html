Programming language

• It is a computer language, programmers use to develop software programs, scripts, the other sets of instructions for computers to execute.

• It is a system of notation for writing computer languages.

• It is an artificial language that can be used to control the behaviour of a machine , particularly a computer.

• It is used to write all computer programs and computer software.

• It is a set of instructions and syntax used to create software programs.

• There are two types of the programming language. i.e. low level language and high level language.
